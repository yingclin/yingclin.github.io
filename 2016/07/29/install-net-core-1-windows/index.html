在 Windows 下安裝 .NET Core 1.0 環境

Jul 29, 2016 | 軟體建構 | Hits

.NET Core 支持多平台下的安裝，包括 Windows、Linux、Mac、Docker。這篇介紹在 Windows 下的安裝方式。

二種安裝方式

安裝 Visual Studio 及配合的套件

官方推薦在 Windows 下最好的開發方式是使用 Visual Studio，下載並安裝免費的 Visual Studio Community 2015 就可輕鬆進行開發。下載並安裝：

  • Visual Studio 2015 Update 3
  • .NET Core 1.0.0 - VS 2015 Tooling Preview 2
    (2016-07-29)

只安裝命令列 SDK

安裝 SDK，在命令列下使用自己愛用的程式（文字）編輯器進行開發。下載並安裝：

  • .NET Core SDK for Windows
  • 可配合微軟出的免費編輯器 Visual Studio Code，對 .NET 的開發有良好的支援。

安裝後測試

確認命令列工具可用

開啟命令提示字元視窗，輸入dotnet --version，可正確執行並顯示版本資訊。

建立並執行簡單專案

在命令提示字元視窗中，執行
mkdir testapp建立工作目錄
dotnet new建立新 console 專案
dotnet restore抓取必要的套件及類別庫
dotnew run執行專案

輸出 (第一次執行會先編譯)
