ASP.NET Core SynchronizationContext

  • ASP.NET Core 移除了 SynchronizationContext。
  • 在 contextless 情況下，await 的接續執行將從 thread pool 開始不用再特別下 ConfigureAwait(false) 了；但做 Library 時還是要下以維持相容性。

USING PROTOBUF MEDIA FORMATTERS WITH ASP.NET CORE

  • Protobuf 是 Google Protocol Buffers 的簡稱，可以把它想像成是 XML 或 JSON 格式的用途 - 介紹文章
  • 透過套件支援和特定 Header： application/x-protobuf，可以快速的實作出 Client/Server 的交互。

RehanSaeed/Schema.NET

  • 要先知道 schema.org - Schema.org 提供了一份共享的詞彙表，站長可以使用它來標記網頁，而這些標記則被主要的搜索引擎： Google， Microsoft， Yandex 和 Yahoo! 所支持。
  • 以物件定義後轉為 JSON-LD 輸出到頁面中；另一個 Microdata 方式，是將資訊放在各別的 tag 中。

系統新增 AutoValidateAntiforgeryTokenAttribute，不用再手動一個個加 [ValidateAntiForgeryToken]，對 GET 和 HEAD 以外的動作全部套用。
