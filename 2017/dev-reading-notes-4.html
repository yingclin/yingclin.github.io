React的新引擎—React Fiber是什么？

  • 介紹了 React Fiber 的目的和做法，用簡單測試來比較差異。
  • React Fiber 帶來的問題
    • task按照優先順序之後，可能低優先順序的任務永遠不會執行，稱之為starvation。
    • task有可能被打斷，需要重新執行，那麼某些依賴生命週期實現的業務邏輯可能會受到影響。
  • React Fiber 帶來的好處
    • 增强了某些領域的支持，比如動畫、佈局和手勢。
    • 在複雜頁面，對用戶的響應會及時，體驗會變好。
  • api 基本上沒有變化，對現有項目很友好。
