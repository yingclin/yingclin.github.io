以 Owin Middleware 實作 Versioning Header

版本資訊，可用自動取得的 Assembly File Version 來表示，並保存起來，避免一直重覆讀取。進階上，可以配合參數，傳入指定的版本資訊或類型，提供更彈性的功能。
