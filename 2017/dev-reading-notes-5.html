.NET Standard 2.0 is final

  • .NET Standard 2.0 is supported on the following platforms:
    • .NET Framework 4.6.1
    • .NET Core 2.0
    • Mono 5.4
    • Xamarin.iOS 10.14
    • Xamarin.Mac 3.8
    • Xamarin.Android 7.5
    • UWP is work in progress and will ship later this year.
  • .NET Framework compatibility mode :
    • found that 70% of all NuGet packages on nuget.org are API compatible with .NET Standard 2.0.
  • .NET Standard, .NET Core 進到 2.0 版可用性提高很多，近來大多數的文章也都是針對 .NET Core 及 ASP.NET Core 做介紹，時機上已經可以開始來完整學習 .NET Core 的使用了。
