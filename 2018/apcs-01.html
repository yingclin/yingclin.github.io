新手的 APCS 速成班實驗記錄 - 01

Sep 7, 2018 | 教學 | Hits

何謂 APCS，官網上的說明如下：

APCS為Advanced Placement Computer Science的英文縮寫，是指「大學程式設計先修檢測」。其檢測模式乃參考美國大學先修課程（Advanced Placement，AP），與各大學合作命題，並確定檢定用題目經過信效度考驗，以確保檢定結果之公信力。

這是一個試辦項目，但它的成績已有部份資訊相關科系在大學個人申請時列入採計或參考了。

下次的考試就在 10/27，怎麼在這麼短的時間內幫助一個高中文組女生利用課餘時間從頭學習，有機會在檢測中取得還過得去的成績，是個死馬當活馬醫有趣的過程。

就算最後沒辦法考到達標的成績，也是一個很好的學習經驗。或許在申請入學時，可以多一點點幫助。

認識 APCS

檢測內容

檢測分為二節次的「程式設計觀念」與一節次的「程式設計實作」，兩科均以中文命題，採線上方式進行測驗。

觀念題

兩份測驗題本共計50道單選題。以運算思維、問題解決與程式設計觀念測試為主。測試包含 code tracing, code completion, code debugging 等題型。

實作題

一份測驗題本共計4個題組，每個題組各有10-20組測試資料。測試方式就是依著題組的說明實作出程式或副程式，可自行選擇以 C, C++, Java, Python 撰寫程式，並正確執行測試資料。題目範例如下：

10月的檢測只有考實作題的部份，所以後續的輔導會以完成實作題為目標。

檢測環境

官方提供虛擬主機的練習環境。

  • 一份 pdf 的說明文件
  • VirtualBox 下載連結
  • 整合好 APCS 開發環境的 Linux Live ISO

實作題可以用 C/C++, Python, Java 來做答，所以環境中提供了這三種語言的開發環境。

答題時，要上電腦答題。
實作題有下載題目及上傳答案的操作，可能會增加錯誤的機會及時間的消耗，要注意。

程式語言

  • Java 1.8.0_171、
  • Gcc 7.3.0 (Ubuntu 7.3.0-16ubuntu3)、
  • C++ 7.3.0 (Ubuntu 7.3.0-16ubuntu3)、
  • Python 2.7.15、Python 3.6.5

IDE

  • Eclipse Oxygen 3a Release(4.7.3a) (Java)
  • Code::Blocks 16.01 (C/C++)
  • Python IDLE 2.7.15、Python IDLE 3.6.5

所有資訊都從官網來的，請以官網最新版本為準，就不加上各項目的連結了。
