新手的 APCS 速成班實驗記錄 - 02

Sep 13, 2018 | 教學 | Hits

這次實驗的目的，就是看能不能在短時間內在輔導一位高中文組女生利用課餘時間學習後，通過 APCS 的實作基本門檻，
選用的程式語言為 Java。

此為記錄的第 2 篇。

輔導策略

現階段的策略，就是利用短時間來學習基本語法及必要(如讀寫文字檔案)的 API 用法後，透過各種練習題及官方提供的考題範例來熟悉。

第一課

建立開發環境

把官方的練習環境建立起來讓學員玩看看。

  • 下載 VirtualBox 和 ISO 檔
  • 安裝 VirtualBox 並依官方說明建立虛擬機
  • 跑起來，玩玩裡面的 eclipse

完看官方環境後，還是在本機電腦上安裝 JDK 8 及 eclipse 才不用每次要練習都要開 VM。

認識 Java

簡單介紹一下 Java。

  • 歷史就先不用說了
  • 介紹一下語言的特性
    • JVM
    • 物件導向的(Object-oriented)
    • 跨平台(Platform- independent)
  • 說明一下物件導向的觀念

類別和物件

以遊戲畫面中的人物和植物為例，配合簡單的類別實作說明類別和物件的差別及類別繼承的概念。

類別的組成

建一個 User 類別說明類別內容的主要組成。

User.java

說明 public 和 private 的差別。

類別(靜態)變數，屬性(成員變數)和區域變數的區別要說明清楚。

執行結果

變數和資料型別

變數的宣告

  • 簡介一下變數的觀念
  • 宣告語法
    type identifier = [value];
    type identifier = new Class();
  • 變數名的命名原則

認識資料型別

  • Primitive Type 和 Reference Type - 物件
  • 常用基本型態(Primitive Type)
    int, long, float, double, boolean
  • String 字串
  • 晉升(Promotion)與轉型(Casting)

宣告一些資料型別執行看看。

算術運算子

數學中的加(+)、減(-)、乘(x))、除(÷)等基本的運算在 Java 中是如何表達及使用。

  • 加法 + => num1 + num2
  • 減法 - => num1 - num2
  • 乘法 * => num1 * num2
  • 除法 / => num1 / num2
  • 模數(餘除) % => num1 % num2

遞增與遞減運算子

變數值加 1 或減 1 在傳統程式中經常被用到，所以系統有提供專用的運算子以加速開發。

  • 遞增運算子: ++
  • 遞減運算子: --

範例：

變數 a 要加 1 時，基本寫法為 a = a + 1，改用遞增運算子為 a++。

練習題

練習01

建一個新專案，在螢幕顯示 “Hello APCS”.

要求

  • 專案名稱 APCS01
  • package ex01

練習02

變數 a 的值為 10,變數 b 的值為 4.
a + b 的結果為 r1, a - b 的結果為 r2, a x b 的結果為 r3, a ÷ b 的結果為 r4.

輸出 a, b, r1, r2, r3, r4 的值到螢幕上.

要求

  • 專案名稱 APCS02
  • package ex02

學員第一次動手建立專案及新增類別時，還是會有沒信心卡住不通的時候，需要在旁邊適時的指導。

以上，開工吧！
